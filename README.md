# Tweeter Project

Tweeter is a simple, single-page Twitter clone.

This repository is the project based on HTML, CSS, JS, jQuery and AJAX front-end skills, and their Node, Express and MongoDB back-end.

## Dependencies
- Express
- Node 5.10.x or above
